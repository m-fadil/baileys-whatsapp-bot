## How to use

1. Clone this repository
2. Install dependencies `npm install`
3. Copy .env.example to .env and update the configuration for `DATABASE_URI` and `DATABASE_NAME`
4. Build `npm run build`
5. Run `node --watch dist/index`

Enjoy~
